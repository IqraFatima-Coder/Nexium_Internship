## Features:
- Header with app name and user controls
- Ingredient management section with visual icons
- Appliance selection checkboxes
- Dietary preference radio buttons
- Cuisine type selection
- Prominent CTA button for recipe generation

## User Flow:
1. User adds/edits ingredients
2. Selects available appliances
3. Chooses dietary preferences
4. Optionally selects cuisine type
5. Clicks "Generate Recipe Ideas"
6. Navigates to recipe output page
